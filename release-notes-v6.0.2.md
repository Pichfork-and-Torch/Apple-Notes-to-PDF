# Apple Notes to PDF v6.0.2

Export Apple Notes to PDF, HTML, and plain text on macOS — local, no cloud upload.

## Install

1. Download the release zip
2. Keep the .app, export-apple-notes.sh, and render-note-to-pdf.applescript together
3. Double-click the app (or run ./export-apple-notes.sh --limit 5)
